# Ligare Call Center

Enterprise-grade call center monorepo for Ligare BPO.

## Structure
- `apps/web` React + TypeScript + Vite
- `apps/api` Java 21 + Spring Boot + WebSocket
- `packages/shared` shared contracts/OpenAPI artifacts
- `scripts` dev/test/e2e automation

## Quick start
```bash
./scripts/dev-up.sh
./scripts/test-all.sh
```
